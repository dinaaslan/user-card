# user-card
For this assignment, I created a user profile card by structuring the card using a main div with the class (card) that contains all the elements of the profile. Inside the card, I added a (top) section to create a colored header and a (content) section to hold the user information. I included a profile picture, the name, location, professional description, and an “edit profile” with icons beside some of these elements to replicate the provided example. 
In the Style sheet, I styled the card by setting a fixed width and height for (.card), centered it on the page using margin:auto, and added a gray border. I use display:flex with flex-direction: column to organize the card sections vertically. Then I styled the .top class with a blue background color. 
For the content; I added spacing using padding and margins and applied sans-sarif font. I also styled the profile picture by making it circular using border-radius and positioned it using position: absolute so it overlaps with the top section. Finally, I resized the icons and added spacing between the icons and text.
